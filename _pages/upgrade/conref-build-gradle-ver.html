marklogic-data-hub

In your build.gradle file, replace all occurrences of your old DHF version number with ``.

  **Example:** In the `plugins` section and the `dependencies` section,

  ```
  plugins {
      id 'net.saliman.properties' version '1.4.6'
      id 'com.marklogic.ml-data-hub' version ''
  }
  ...
  dependencies {
    compile 'com.marklogic:marklogic-data-hub:'
    compile 'com.marklogic:marklogic-xcc:9.0.6'
  }
  ```
